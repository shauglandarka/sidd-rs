Simple crate for SIDD I/O

This is only set up for SIDD 1.0, version 2.0 will come out when I need it.

Only reads one image segment for now. So keep the SIDD smaller than ~9.8 GB
and keep it MONO8I.

See the example directory for how to use this crate.

Please let me know any problems, don't trust the code!
